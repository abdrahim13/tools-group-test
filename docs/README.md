# ToolsGroup Todo App



#### Docker Build Script (/backend)

```bash
  cd /backend
  ./build_docker.sh
```



## Frontend

#### Development

```bash
npm run dev
```

#### Build

```bash
npm run build
```
#### Note : app Already build and served by UWSGI python server  




## Backend

#### Development

```bash
  python3 manage.py run
```






## Database

#### Migrate

```bash
  python3 manage.py create_db
```
